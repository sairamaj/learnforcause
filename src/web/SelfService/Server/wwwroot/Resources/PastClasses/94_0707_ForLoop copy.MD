#### Things learned today
* For loop
    * used for repetition
        * start value (ex: x=10)
        * condition (to stop) (till x<10)
        * increment ( ex: x=x+2)
    * Syntax
    ```java
        for(int x=0;  ; x<10   ; x= x+1){

        }
    ```
